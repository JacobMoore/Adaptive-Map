Wrench

A wrench consists of a single force vector and a single moment vector which are collinear or parallel.

A "wrench" in engineering statics is a special type of force/couple system where the force vector and the moment vector of the couple are parallel. An example of a wrench on a real world system is the force and moment exerted by a screwdriver. A person using the screwdriver both pushes down along the shaft of the screwdriver and exerts a moment (from a couple) about the axis of the screwdriver. Any wrench where the force and moment vectors have the same direction is considered a positive wrench. Conversely, a wrench where the force and moment vectors have opposite directions is considered a negative wrench.

Converting a Force Couple System into a Wrench

Any force couple system that is not already a wrench can be converted to a wrench by changing the point of application of the force. If we think about the some equivalent force/couple system where the force and moment vectors are not parallel (as shown in A in the diagram below), there is a component of the moment vector that is parallel with the force vector and a component that is perpendicular to the force vector (as shown in B). Now, if we choose a new point of application out along a line perpendicular to both components of the moment vectors, this force will exert a moment about the original point of application in the direction of the component that is perpendicular to the force (as shown in C). By matching the moment vector this new moment (d cross F) and the perpendicular component of the old moment vector, we can find the point at which there will be no perpendicular component of moment vector. At this new point of application, the force/couple system is a wrench (as shown in D) with a moment that is parallel to force.

Now we need to move the point of application somewhere along the line perpendicular to both the force vector and the moment vector. Since both of these vectors lie in the XY plane, the line perpendicular to both of them is in the Z direction.
