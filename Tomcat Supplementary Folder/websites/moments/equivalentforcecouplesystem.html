Equivalent Force Couple System

If we have some set of forces and/or couples acting on a body (like in the diagram below) we learned how to resolve each of those forces into a force and a couple (the translational and rotational components respectively). If all of these resolved forces are concurrent (which happens if we resolve all the forces with respect to the same point) we can simply add them up to find a single equivalent force. Similarly, the point of application of couples can be moved without changing the system. We can therefore simply add up all the couple vectors to find a single equivalent couple. This means that any set of forces and/or couples acting on a body can be reduced to a single equivalent force and a single equivalent couple. With only a single force and a single couple, this system is the simplest statically equivalent system to the original system.

Some forces and couples acting on a body.

To find the equivalent force couple system for any set of forces, you can use the following steps.

  1. First decompose all the forces into forces and couples. Choose a common point (usually the center of mass or some anchored point) and have all the decomposed forces act on that point.
    2. The each force is decomposed into a force and a couple.
  2. Now with all the forces acting on a single point, you can add all of these forces by adding the force vectors. The sum of all the force vectors is the 'force' in the 'equivalent force couple system.
    3. \[ \mathrm{ \mathbf{F_{EQ}} = \mathbf{F_1} + \mathbf{F_2} + \mathbf{F_3} + \dots }\]
  3. All of the couples can also be added up by adding the moment vectors of the couples. This includes the couples originally exerted on the system and the couple components of the forces. Since the point of application of couples can be moved without changing the system the point of application of the couples is unimportant. The sum of all the moment vectors of the couples is the 'couple' of the 'equivalent force couple system'. Traditionally, the couple vector is drawn acting on the same point as the force vector.
    4. \[ \mathrm{ \mathbf{M_{EQ}} = \mathbf{M_1} + \mathbf{M_2} + \mathbf{M_3} + \dots }\]
The equivalent force couple system for the original set of forces and couples.
