Equivalent Point Load (Distributed Forces)

An equivalent point load is a single point force that will have the same effect on a body as the original loading condition, which is usually either 1) several point forces or 2) a distributed load. The equivalent point load should always cause the same linear acceleration and angular acceleration as the original force or forces it is equivalent to. Finding the equivalent point load for a number of other point forces or a distributed force often helps simplify the analysis of a system. On this page we discuss the process of finding the equivalent point load for distributed forces.

F3 will have the same effect on the block as F1 and F2 together. F3 is the equivalent point load for the system with F1 and F2.
F3 will have the same effect on the block as the distributed force F1. F3 is the equivalent point load for the distributed force F1.

Finding the Equivalent Point Load for Distributed Forces

The method for finding the equivalent point load for a distributed force builds upon the process used for finding the equivalent point load for multiple parallel point forces. A distributed force can just be thought of as an infinite number of point forces each with a very small magnitude. To find the equivalent point load for a distributed force, we will modify the equations we used for parallel point forces so that they will use integration rather than summation.

2-D Distributed Forces with a Single Parallel Direction

We will start with the simplest example, a 2-D problem where the point of application can be described by a single number (x) and the direction of the distributed force is a single uniform direction.

An example of a distributed load. The magnitude of the force is given as a function of x.

Magnitude: To find the magnitude of the equivalent point load for a collection of parallel point loads we simply added up all the magnitudes. To find the magnitude for a distributed force, we integrate the force function along the entire length that the force is exerted over. The formula below is the formula for the magnitude of the equivalent point load of a 2-D distributed force with a single uniform direction.

\[ \mathrm{ F_{eq} = \int_{min}^{max} F(x) * d(x) }\]

where min is the minimum value of x (where the force begins) and X is the maximum value of x (where the force ends).

Direction: The direction of the equivalent point load is the same as the direction of the distributed force in this case since the distributed force has a single uniform direction.

Point of Application: To find the point of application for our equivalent point load, we modify the point of application formula for parallel point forces. Similar the point forces formula, this equation ensures that the equivalent point load exerts the same moment on the body that the distributed force does.

\[ \mathrm{ x_{eq} = \frac{\int_{min}^{max} F(x) * x * d(x)}{F_{eq}} }\]

The formula for \( \mathrm{x_{eq}} \) shown above is also the formula for the x coordinate of the centroid of the area under the function F(x). This means that all the tricks that can be used to find the centroid of an area (mainly using composite parts) can be used to find the point of application of the equivalent point load for a situation like the one above.

3-D Distributed Forces with a Single Parallel Direction

Next we will increase the complexity of the problem by adding another dimension. Now instead of assuming we have a force distributed over a line, we have a force distributed over an area. For this new distributed force, the magnitude will depend upon two dimensions F (x, y). A force distributed over an area might look like the example below.

In this example of a force distributed over an area, the force is described by the function F (x, y) = (5x + 10y) \( \mathrm{\frac{N}{m^2}} \). The area of interest (shown in blue) is a circular area one meter in diameter. This problem represents the force on a circular hatch covered in an uneven layer of sand.

Magnitude: The magnitude of the equivalent point load is again found through integration. For a 3-D problem like this though we are integrating for the volume under the surface F (x ,y) with respect to the area.

\[ \mathrm{ F_{eq} = \int F(x, y) * dA }\]

To evaluate this integral, we will have to perform a double integral evaluating the integral with respect to one variable at a time (integrating with respect to x and then y or vice versa). Below is the formula integrating the function F(x, y) with respect to x and then with respect to y.

\[ \mathrm{ F_{eq} = \int F(x, y) dA = \int_{ymin}^{ymax} \left[\int_{\text{xmin at y}}^{\text{xmax at y}} F(x, y) * d(x)\right] d(y) }\]

The variables "ymin" and "ymax" represent the y extents of the area you are looking at (the min and max y values for the area of the circular porthole in the example). The variables "xmin at y" and "xmax at y" represent the minimum and maximum x value of the area for each value of y. These will be functions, giving the maximum or minimum value of x in terms of y. Also remember that when integrating with respect to one variable (such as x) all other variables (such as y) should be treated as constants.

Direction: The direction is once again simple, the distributed force has a single uniform direction the equivalent point load will have that same direction.

Point of Application: Now that we have an area, we need to solve for two coordinates for the point of application of the equivalent point load. In the example above, we need to solve for an x and a y coordinate. We use the following formulas to solve for the x and y coordinates of the equivalent point load. Like in the 2-D case, we are ensuring that the distributed force and the equivalent point load exert the same moment on the body.

\[ \mathrm{ x_{eq} = \frac{\int\left( x *F(x, y) * dA\right)}{F_{eq}} }\] \[ \mathrm{ y_{eq} = \frac{\int\left( y *F(x, y) * dA\right)}{F_{eq}} }\]

Like with the magnitude, we need to transform the integrals to double integrals to solve the equations.

\[ \mathrm{ x_{eq} = \frac{\int_{ymin}^{ymax} \left[\int_{\text{xmin at y}}^{\text{xmax at y}}\left( x * F(x, y) * d(x)\right)\right] d(y)}{F_{eq}} }\] \[ \mathrm{ y_{eq} = \frac{\int_{ymin}^{ymax} \left[\int_{\text{xmin at y}}^{\text{xmax at y}}\left( y * F(x, y) * d(x)\right)\right] d(y)}{F_{eq}} }\]
Distributed Forces without a Single Parallel Direction

To model systems with non-parallel distributed forces, it is necessary to break the force down into its components. Below is an example of a non-parallel distributed force acting on a curved surface.

This image shows the pressure of water pressing up against a cross section of a dam that is ten meters wide and has a quarter circle cross section.

If we think of the distributed force as a large collection of point forces, each one of those forces would have a horizontal component and a vertical component. To analyze non-parallel distributed forces, we need to write out an equation for the magnitudes of each component of these vectors. Below the horizontal and vertical components of the above example are separated. Each component of the distributed force is then a set of parallel vectors.

The component functions of a distributed force.

Once we have the functions for the magnitude of each component of the force, we can deal with each component separately. In our example, which is a 2-D problem, we have two distributed forces each with a single uniform direction. Each one of these can be analyzed as a parallel distributed force and an equivalent point load can be found for each. After an equivalent point load is found for each component, an equivalent point load can be found for the collection of point forces. Note that the process of the functions of the components and then integrating these components is not a trivial task and may require computer tools to solve all but the simplest of problems.

3-D distributed non-parallel forces are similar in nature. They can be broken down into 3 separate 3-D parallel distributed forces. Each one of components can be analyzed as a separate 3-D parallel distributed force using the processes above and will have an equivalent point load. These point loads can be combined find the overall equivalent point load.

Worked Problems

Worked Problem 1:

Find the equivalent point load for the distributed force shown below. Be sure to report the magnitude, direction and point of application of the equivalent point load.



Work Comments
\[ \mathrm{ F_{eq} = \int_{min}^{max} F(x) * d(x) }\] \[ \mathrm{ F_{eq} = \int_0^5 (x^2 - 10x +25) d(x) }\] \[ \mathrm{ F_{eq} = \bigg|_0^5 \frac{x^3}{3} - \frac{10x^2}{2} +25x }\] \[ \mathrm{ F_{eq} = \left(\frac{5^3}{3} - \frac{10*5^2}{2} +25*5\right) - \left(\frac{0^3}{3} - \frac{10*0^2}{2} +25*0\right) = 41.67 \text{ N} }\]

The first step is to solve for the magnitude of the equivalent point load \( \mathrm{(F_{eq})} \). To do this we are integrating the function F(x) from the start to the end of the beam (0 to 5 meters). This is a definite integral, which means that there is no constant of integration (C)
\[ \mathrm{ x_{eq} = \frac{\int_{min}^{max} x * F(x) * d(x)}{F_{eq}} }\] \[ \mathrm{ x_{eq} = \frac{\int_{0}^{5} (x^3 - 10x^2 +25x) * d(x)}{41.67} }\] \[ \mathrm{ x_{eq} = \frac{ \bigg|_0^5 \frac{x^4}{4} - \frac{10x^3}{3} + \frac{25x^2}{2}}{41.67} }\] \[ \mathrm{ x_{eq} = \frac{ ( \frac{5^4}{4} - \frac{10*5^3}{3} + \frac{25*5^2}{2} ) - ( \frac{0^4}{4} - \frac{10*0^3}{3} + \frac{25*0^2}{2} )} {41.67} = 1.25 \text{ m} }\] Next we solve for the point of application of the equivalent point load \( \mathrm{(x_{eq})} \). To do this, we are integrating x * F(x) over the length of the beam and then dividing by the magnitude of the equivalent point load. This ensures that the moment exerted by the equivalent point load is the same as the moment exerted by the distributed force.
After we have calculated the magnitude and point of application (and knowing the direction is -Y) we can draw out a diagram of \( \mathrm{F_{eq}} \) acting on the beam.

Worked Problem 2:

The wind has piled up sand into a corner on a building. The building supervisor is worried about the weight of the sand pushing against the roof of the basement below. The function describing force of the sand pushing down on the surface is given below. Find the equivalent point load for the distributed force of the sand. Be sure to report the magnitude, direction and point of application of the equivalent point load.



Work Comments
\[ \mathrm{ F_{eq} = \int F(x, y) dA = \int_{ymin}^{ymax} \left[\int_{\text{xmin at y}}^{\text{xmax at y}} F(x, y) * d(x)\right] d(y) }\] \[ \mathrm{ F_{eq} = \int_{ymin}^{ymax} \left[\int_{\text{xmin at y}}^{\text{xmax at y}} (30 - 5x - 10y) * d(x)\right] d(y) }\]

The first step in solving for any equivalent point load is to solve for the magnitude of the equivalent point load. In this case, we have a force distributed over an area with the force as function of both x and y. We use the formula to the left to solve for the magnitude. Knowing the formula, we can plug values into the formula.


\[ \mathrm{ F_{eq} = \int_0^3 \left[\int_0^{x=6-2y} (30 - 5x - 10y) * d(x)\right] d(y) }\] 		Next we need to solve for "ymin", "ymax", "xmin at y" and "xmax at y". The values for "ymin" and "ymax" will each be a single number indicating the y extents of the area the force is applied to. The values for "xmin and y" and "xmax at y" will be functions outlining the area of application of the force. After we find these values we plug them into our formula.
\[ \mathrm{ F_{eq} = \int_0^3 \left[\bigg|_0^{x=6-2y} (30x - \frac{5}{2}x^2 - 10xy)\right] d(y) }\] \[ \mathrm{ F_{eq} = \int_0^3 \left[\left[(30(6-2y) - \frac{5}{2}(6-2y)^2 - 10(6-2y)y)\right] - \left[ 0 \right]\right] d(y) }\] \[ \mathrm{ F_{eq} = \int_0^3 \left[ 10y^2 - 60y + 90 \right] d(y) }\] \[ \mathrm{ F_{eq} = \bigg|_0^3 \frac{10}{3}y^3 - 30y^2 + 90y }\] \[ \mathrm{ F_{eq} = \left[\frac{10}{3}(3)^3 - 30(3)^2 + 90(3)\right] - \left[ 0 \right] }\] \[ \mathrm{ F_{eq} = 90 \text{ lbs} }\]

Next we evaluate the integrals and solve for \( \mathrm{(F_{eq})} \). Remember when we are intergrating with respect to x, we assume that y is simply a constant.
\[ \mathrm{ x_{eq} = \frac{\int_{ymin}^{ymax} \left[\int_{\text{xmin at y}}^{\text{xmax at y}}\left( x * F(x, y) * d(x)\right)\right] d(y)}{F_{eq}} }\] \[ \mathrm{ x_{eq} = \frac{\int_0^3 \left[\int_0^{6-2y}\left( x * (30 - 5x - 10y) * d(x)\right)\right] d(y)}{90} }\] \[ \mathrm{ x_{eq} = \frac{\int_0^3 \left[\int_0^{6-2y}(30x - 5x^2 - 10yx) * d(x)\right] d(y)}{90} }\] \[ \mathrm{ x_{eq} = \frac{\int_0^3 \left[\bigg|_0^{x=6-2y} \left( 15x^2 - \frac{5}{3}x^3 - 5yx^2 \right)\right] d(y)}{90} }\] \[ \mathrm{ x_{eq} = \frac{\int_0^3 \left[ \left( 15(6-2y)^2 - \frac{5}{3}(6-2y)^3 - 5y(6-2y)^2 \right)-(0)\right] d(y)}{90} }\] \[ \mathrm{ x_{eq} = \frac{\int_0^3 \left[ -\frac{20}{3}y^3 + 60y^2 -180y +180 \right] d(y)}{90} }\] \[ \mathrm{ x_{eq} = \frac{ \bigg|_0^3 \left[ -\frac{5}{3}y^4 + 20y^3 -90y^2 +180y \right]}{90} }\] \[ \mathrm{ x_{eq} = \frac{ \left(-\frac{5}{3}(3)^4 + 20(3)^3 -90(3)^2 +180(3) \right)-(0)}{90} }\] \[ \mathrm{ x_{eq} = 1.5 \text{ ft} }\]

Now that we have \( \mathrm{(F_{eq})} \), we can solve for the location of the point of application. We will start with the x coordinate (\( \mathrm{x_{eq}} \)). We've already solved for most of the values we need to plug in with the prior calculations, so the formula basically just needs to be solved.
\[ \mathrm{ y_{eq} = \frac{\int_{ymin}^{ymax} \left[\int_{\text{xmin at y}}^{\text{xmax at y}}\left( y * F(x, y) * d(x)\right)\right] d(y)}{F_{eq}} }\] \[ \mathrm{ y_{eq} = \frac{\int_0^3 \left[\int_0^{6-2y}\left( y * (30 - 5x - 10y) * d(x)\right)\right] d(y)}{90} }\] \[ \mathrm{ y_{eq} = \frac{\int_0^3 \left[\int_0^{6-2y}\left( 30y - 5xy - 10y^2 \right) * d(x)\right] d(y)}{90} }\] \[ \mathrm{ y_{eq} = \frac{\int_0^3 \left[left[\bigg|_0^{x=6-2y}\left( 30yx - \frac{5}{2}yx^2 - 10y^2x \right) \right] d(y)}{90} }\] \[ \mathrm{ y_{eq} = \frac{\int_0^3 \left[left[ \left( 30y(6-2y) - \frac{5}{2}y(6-2y)^2 - 10y^2(6-2y) \right)-(0) \right] d(y)}{90} }\] \[ \mathrm{ y_{eq} = \frac{\int_0^3 \left[ 10y^3 -60y^2 + 90y \right] d(y)}{90} }\] \[ \mathrm{ y_{eq} = \frac{ \bigg|_0^3 \left[ \frac{5}{2}y^4 -20y^3 + 45y^2 \right]}{90} }\] \[ \mathrm{ y_{eq} = \frac{ \left( \frac{5}{2}(3)^4 -20(3)^3 + 45(3)^2 \right) - (0)}{90} }\] \[ \mathrm{ y_{eq} = .75 \text{ ft} }\]

Now we repeat the process for the y coordinate (\( \mathrm{y_{eq}} \)).


Now that we have solved for the magnitude and the point of application of the equivalent point load, we can draw a diagram showing the magnitude, direction and point of application of the equivalent point load for the sand accumulated in the corner.
