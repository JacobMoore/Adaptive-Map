Troubleshooting

If you are having trouble running the Adaptive Map Textbook, this page outlines some known issues and the solutions to those problems. We are striving to create a tool that is accessible to everyone and stable across computers, but the tool is still under development so there are still bugs to fix. We apologize for the inconvenience and we thank you for your patience.

The recommended steps to resolve issues are listed below. It is recommended that you try the following in order.

  1. Browsers: We have the tool working in the Internet Explorer, Firefox, and Chrome browsers. There are known issues with the Safari browser that we are working to resolve. For now it is recommended that you do not use the Safari web browser.

  2. Check your version of Java: The core code for the adaptive map tool is a Java Applet. To run the tool, you need Java version 6 or later. To check your version of Java you can go to www.java.com and click on the “Do I Have Java?” link. If you have a version 5 or earlier of Java, you will need to update. To update Java, go back to www.java.com and click on the “Free Java Download” to download and install the latest version of Java.

  3. Other Problems: Try clearing your cache so that the applet is re-downloaded. On Firefox, go to History->Clear Recent History, and select Cache. Then re-visit the page.

  4. If you are still having issues, please feel free to e-mail me at moorej7@vt.edu. Provide as many details as possible on the error (browser, error messages if you are getting them, and what is showing up on the screen) and I will work to resolve the issue. Any information we get on software issues will help us make the tool better.
